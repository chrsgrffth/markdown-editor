# Markdown Editor

Built on [Vue 2x](https://github.com/vuejs/vue) with [Decent](https://github.com/octopuscreative/decent-scss).  

<br>
## Building
#### Development
```
$ npm install
$ npm start
```

#### Production
```
$ brunch build --production
```
